Nano Photo editor

Anipong has been upgraded to the more powerful "Nano Photo" app.
The "Nano Photo" app uses Google’s Nano Banana to enable powerful image generation and editing through prompts alone. It also integrates all the features of the previous "Anipong" app and offers a new UI/UX experience.

The new Nano Photo app supports two modes by adding Nano Banana to the existing Anime Style conversion. Enjoy fun anime transformation images easily with Nano Photo Style mode that supports 12 style conversions including popular anime, and make your photos an infinite creative canvas with prompt input in Nano Photo Edit mode.

Nano Photo's innovative slot image system provides a more convenient multi-editing environment.

The "Nano Photo" app v2.0.3 is currently under review on the App Store. For this reason, only the existing "Anipong" app v1.1.4, which supports anime conversion without Nano Banana, is currently available.
The iOS version of "Nano Photo" v2.1.3, which includes Nano Banana and introduces a new UI/UX, will be released as soon as possible.
The Android version of "Nano Photo" v2.1.3 is already available on the Play Store.
